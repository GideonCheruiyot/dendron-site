## Summary
- type: `boolean`

## Description

{{fm.desc}}.

If you set `todoIntegration` to true, task notes will create a `TODO:` keyword in the frontmatter that should be picked up by TODO tree.

## Example

```yaml
workspace:
    task:
      todoIntegration: true
```

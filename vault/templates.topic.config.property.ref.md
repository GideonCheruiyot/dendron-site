## Fields

### default

### required

### noteOverride
Can you override this property via the note frontmatter?

- NOTE: currently, all note overrides are in the following form (in the future, we might split up the override between publishing and preview)
```yml
config:
  global:
    {nameOfOverride}: ...
```

- type: `boolean`
- default: `true` 
- required: `false`

## Description
{{fm.desc}}

This also applies to note references. 

If set to `false`, the entire hierarchy string will be used for rendering the links.

## Example

```yml
  preview:
    enableNoteTitleForLink: true
```

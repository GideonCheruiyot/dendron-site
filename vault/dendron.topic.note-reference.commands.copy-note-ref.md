-   shortcuts:
    -   key: `ctrl+shift+r`
    -   mac: `cmd+shift+r`
    -   when: `editorFocus`

Copies a reference to the current open document

This lets you quickly create a [[note reference|dendron.topic.note-reference]] to the current note.
